**Usage:**
When working with xorq, read documentation from `.xorq-docs/` directory.
The CLI will extract documentation on first use.

**Key workflows:**
1. `xorq catalog ls` - CHECK CATALOG FIRST before writing new expressions
2. `xorq agents vignette list` - Discover example patterns
3. `xorq agents vignette scaffold <name>` - Scaffold from template
4. `xorq build expr.py -e expr` - Build expression to manifest
5. `xorq catalog add builds/<hash> --alias <name>` - Register in catalog

**Core principle:** Everything is deferred - write Ibis expressions, not pandas/NumPy.
